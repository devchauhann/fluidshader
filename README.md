# FluidShader - Free WebGL Fluid Gradient Editor

![FluidShader](https://fluidshader.vercel.app/og-image.jpg)

A powerful, free web-based tool to create stunning fluid gradient animations with real-time WebGL shader preview. Generate production-ready shader code instantly with customizable colors, speed, warp intensity, noise, and grain effects.

## 🎨 Features

- **Real-Time Preview** - Watch your gradient animations update instantly as you adjust parameters
- **Customizable Colors** - Choose from 5 color channels (cyan, yellow, orange, purple, blue)
- **Parameter Control** - Fine-tune:
  - Flow Speed (0.0 - 0.5)
  - Warp Bloom (0.0 - 0.6)
  - Noise Scale (0.5 - 4.0)
  - Grain Film (0.0 - 0.15)
- **One-Click Copy** - Copy production-ready WebGL shader code to your clipboard
- **Save Designs** - Persist your favorite shader configurations locally
- **Curated Examples** - Browse 8 pre-built shader presets
- **Mobile Optimized** - Full responsive design for all devices
- **No Installation** - Works directly in your browser

## 🚀 Quick Start

Visit **[FluidShader.vercel.app](https://fluidshader.vercel.app/)** and start creating!

### How to Use

1. **Adjust Colors** - Click color circles in the sidebar to change gradient colors
2. **Tune Parameters** - Use sliders to adjust flow speed, warp, noise, and grain
3. **Preview** - See changes in real-time on the canvas
4. **Export** - Click "Copy shader code" to get production-ready WebGL code
5. **Save** - Click "Save Design" to persist your creation
